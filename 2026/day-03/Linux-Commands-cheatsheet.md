### Linux Commands:



#### Process Management commands



ps - 	This command will show you the information about the programs that are currently running on your computer.



top - 	This command will lets you see how much of the computers resources (like CPU and memory) are being used and as well as the details about the running 	programs and processes.



htop - 	This command use to shoe the processes in Interactive process viewer and more friendly than top command.



kill - 	This command wil allow you to stop the running program by sending it a signal.



psaux - This command will display a detailed list of all the running programs on the system including those started by the users.



systemctl - This command will manage the system services and the processes into the your system.



top -H - This command will show the real time system resource usage like CPU, memory, and the details about running programs in a hierarchical view.



lsof - 	This command will lists open the files and the programs that are using onto them.This command will lists open the files and the programs that are 	using onto them.

#### 3\. Networking commands in linux



ifconfig - Shows and configures the network interfaces on the current system.



ping - 	Tests network connectivity between a host/client and server by sending data packets and measuring the response time.



route - Configures the kernel's routing table entries.



traceroute - Displays the path and hops that packets take to reach a remote server.



telnet - Establishes the connection to the remote server or system through a TCP/IP network using the Telnet protocol.



nslookup - Performs DNS lookups to resolve hostnames, IP addresses, mappings, or any specific DNS record.



dig - Performs DNS lookups to query information about DNS name servers.
